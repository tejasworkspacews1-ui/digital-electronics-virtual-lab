# Digital Electronics Virtual Lab

A multi-page, responsive, syllabus-driven virtual lab for Digital Electronics.

## Run locally
Open `index.html` directly, or serve the folder with a lightweight local server:

```bash
python -m http.server 8000
```

Then visit:

```bash
http://localhost:8000
```

## Pages
- Home
- About Us
- Modules
- Simulations
- Contact Us

## Features
- Responsive desktop/mobile navigation
- Theme switcher
- Google Translate widget
- Slideshow carousel
- Horizontal animated gallery
- Interactive logic gate lab
- Boolean expression evaluator
- Arithmetic circuit workbench
- Flip-flop stepper
- K-map and canonical-form builder
- Contact form and professional links
